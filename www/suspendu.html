Le Pendu

Bienvenue, c'est le jeu du pendu. Ce jeu a été réalisé par Lény et Brice. Merci à eux.

Le but du jeu

Le but du jeu est de trouver un mot sans être pendu.

Le déroulement du jeu

Choisissez parmis les lettres disponibles. Si la lettre est dans le mot, elle apparaîtra ainsi, sinon elle appraîtra dans les lettres non présentes dans le mot et vous perdez un tour. Vous avez jusqu'à 12 tours pour éviter d'être pendu et trouver le mot complet.
